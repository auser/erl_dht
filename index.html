erl_dht by saketkunwar

Introduction

 

Erl_dht  is a simulation  framework for evaluating and deploying routing schemes for distributed hash table.Currently chord has been implemented but the modular nature of the core class makes it extensible making it possible to implement other protocols.

An important feature is the ease of use for live deployment by implementing a communication interface.

Erl_dht is coded in erlang otp ,a highly concurrent ,distributed and fault tolerant functional programming language. The message passing feature of erlang is especially suited

for performing simulation.
